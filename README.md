### How it works?
- One master arduino operates on 3 slave ones
- Max 2 fingers could be powered through one arduino
  
### Operational Components:  
- x4 `Arduino Uno`
- x2 `Breadbord`
- x5 `Servo`
- ∞ `Cables`
